# ZenderBox Brand Bible

Guía de identidad visual completa para ZenderBox — servicio de casillero y package-forwarding para latinoamericanos que compran en EE.UU.

## Contenido

| Archivo | Descripción |
|---------|-------------|
| [`index.html`](index.html) | Dashboard visual interactivo del brand bible |
| [`BRAND.md`](BRAND.md) | Guía completa de marca para referencia de IA/código |
| [`tokens.md`](tokens.md) | Tokens de diseño: colores, tipografía, espaciado, radios |
| [`voice.md`](voice.md) | Voz, tono y lineamientos de copy |
| [`photography.md`](photography.md) | Dirección de fotografía, ilustración y estilo visual |

## Cómo usar este repositorio

### Para IAs y agentes de código

Cuando un agente de IA necesite crear, modificar o evaluar cualquier pieza visual o de contenido para ZenderBox, debe apuntar a [`BRAND.md`](BRAND.md) como fuente de verdad principal. Ese archivo contiene todos los tokens, reglas y ejemplos en formato estructurado.

```
Lineamientos de marca ZenderBox:
https://raw.githubusercontent.com/zenderhub/ZenderBox-BrandBible/main/BRAND.md
```

### Para diseñadores y desarrolladores

Abre [`index.html`](index.html) directamente en el navegador — no requiere servidor ni dependencias. Todo está autocontenido.

## Productos cubiertos

Este brand bible cubre la identidad de **ZenderBox** (App móvil + Dashboard web). Para ZenderHub (operador B2B) existe una identidad separada.

## Principio core: White First

El fondo dominante es siempre blanco. Los colores de marca (cyan `#029ECB`, lime `#CCD32A` y el gradiente cyan→lime) se usan con moderación: acciones primarias, estados activos, datos clave, capa de fidelización.
